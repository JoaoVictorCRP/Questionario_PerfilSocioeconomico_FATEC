# Bibliotecas utilizadas:
- Pandas ➡️ Extração de dados ⛏️
- MatPlotLib ➡️ Criação de gráficos 🧮
- TkInter ➡️ Interface gráfica 🖥️

### Todos os conhecimentos a respeito destas bibliotecas foram adquiridos através das respectivas documentações. 

# Visualização dos gráficos 📈

- Para as perguntas abertas foi feita uma nuvem de palavras;
- Erros ortográficos presentes no formulário original foram corrigidos.
